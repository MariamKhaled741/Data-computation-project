# Data-computation-project
This repository contains a comprehensive data science pipeline developed as a final project for the **Data Computation** course at **Alexandria University (FCDS)**. The project focuses on exploring, analyzing, and modeling a large-scale dataset using **Support Vector Machines (SVM)**.

### 🛠️ Project Stages

-   **EDA:** Thorough exploratory data analysis and visualization.
    
-   **Preprocessing:** Data cleaning, handling missing values, and outlier detection.
    
-   **Dimensionality Reduction:** Feature selection/reduction techniques to optimize performance.
    
-   **Modeling:** Robust SVM implementation with hyperparameter tuning.
    
-   **Evaluation:** Comprehensive performance metrics for the final model.
